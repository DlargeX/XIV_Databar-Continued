## [5.6](https://github.com/ZelionGG/XIV_Databar-Continued/releases/tag/5.6) (2026-07-09)

> **Version 5.6 - Equipment Sets, Lockouts & QoL**
>
> This update adds quick equipment set switching to the **Armor** module, saved instance lockouts to the **Clock** tooltip, and audio output selection to **Master Volume**.
> It also improves **Travel** tooltips, consolidates profile sharing in the **Profiles** options tab, and fixes **Vault** and lockout display issues.

### _Global :_

- 🆕 _**NEW** -_ Added saved raid and dungeon lockouts to the **Clock** module tooltip with configurable date/time formats and optional boss progress.
- 🆕 _**NEW** -_ Added audio output device selection to the **Master Volume** module with current device shown in tooltip and Shift+Left-Click popup.
- 🛠️ _**IMPROVEMENT** -_ Consolidated profile import/export into the **Profiles** options tab.
- 🛠️ _**IMPROVEMENT** -_ Added a blank line after section headers in **Travel** module tooltips for consistent layout.
- 🐞 _**BUGFIX** -_ Fixed **Clock** lockout reset times showing off-by-one-minute values and stale data on first tooltip hover after reload.

### _Retail :_

- 🆕 _**NEW** -_ Added quick equipment set switching to the **Armor** module with left-click popup, active set highlighting in tooltip and selection menu and hidden durability display when no slot reports data.
- 🆕 _**NEW** -_ Added shared **Mythic+ Teleports** cooldown display to the **Travel** module tooltip.
- 🐞 _**BUGFIX** -_ Fixed **Vault** frame not closing with the Escape key.
